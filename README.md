# CÁCH DÙNG:
 1. Chọn file Shopee (script.js) hoặc Tiki (tiki.js)
 2. Vào Shopee hoặc Tiki, đăng nhập bằng tài khoản của bạn
 3. Mở Console lên (F12, Ctrl+Shift+C, ...)
 4. Dán đoạn code vào
 5. List item
 6. Gõ "getStatistics()" và ngồi chờ...
